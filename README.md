# React Portfolio Template

A modern, customizable portfolio website built with React and Tailwind CSS.

## Features

- Modern UI with green gradient theme
- Interactive welcome screen
- Projects showcase
- Skills with progress bars
-  Contact form modal
-  Social media links
-  Fully responsive

## Quick Start

### Installation

\`\`\`bash
npm install
\`\`\`

### Development

\`\`\`bash
npm start
\`\`\`

Opens at `http://localhost:3000`

### Build for Production

\`\`\`bash
npm run build
\`\`\`

## Customization

Edit `src/App.js` to customize:

- Your name and title
- Skills and skill levels
- Projects and descriptions
- Social media links

## Technologies

- React 18
- Tailwind CSS
- Lucide React Icons

## Template Usage

This is a template repository. Click "Use this template" to create your own portfolio!
